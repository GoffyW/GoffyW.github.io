包装类

基本类型，引用类型
以对象的形式保存基本类型的值，Java提供了对应的包装类
Byte byte Short short Integer int Long long
Float float Double double Character char Boolean boolean 
int i = 10;
Integer iObj = new Integer(10);

所有的类都是Object类的子孙

所有的对象一定是Object的子类对象

所有的对象可以赋给Object类型的引用类型

  • 一方面出于性能方面的考虑，Java为数值使用基本类型，而不是对象。基本类型不是对象层次的组成部分，它们不继承Object
  • 另一方面有时需要创建表示基本类型的对象，例如集合类只处理对象。为了在对象中存储基本类型，需要将基本类型包装到一个类中，为此Java为8种基本数据类型分别提供了对应的包装类。本质上这些类将基本类型包装到一个类中，因此通常将它们称为类型包装器。包装器类位于Java.lang包中
自动装箱与自动拆箱

举例：

int i = 100;
String str2 = Integer.toString(i);     //第一种方法：利用静态的toString()方法
String str3 = String.valueOf(i);       //第二种方法：利用String类中的valueOf()方法
String str4 = i + "";                 //第三种方法：投机取巧的方法
Integer objI = i;                      //先自动装箱
String str1 = objI.toString();         //第四种方法：先装箱，再利用toString()方法
System.out.println(str1);
System.out.println(str2);
System.out.println(str3);
System.out.println(str4);
